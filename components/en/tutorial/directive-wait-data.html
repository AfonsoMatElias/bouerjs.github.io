Directives are basically attributes that changes some element behavior, and in Bouer there are some directive that we can use to achieve specific behaviors... We are going to list and explain some of them.

Waiting for Data

Using the wait-data directive we are able to hold elements compilation untill we provide data to them.

Telling to the compiler to wait 


     Content

Providing the data that should be used to compile the element 


    app.$wait.set('data-key', { /* ... */ });

In the illustrated example, there is a nice covered background with a bit of wave on it, this is called skeleton and it's can be a achieved by defining e-skeleton directive in the element that we wan't to have this nice effect. For further information check Skeleton Details.
