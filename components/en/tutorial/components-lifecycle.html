Every component has a lifecycle that starts when it is created and ends when it's destroyed. There are a handful of functions that allow you to run code at certain moments during that lifecycle.

We already faced some of them, like loaded and destroyed.

In the component <script>, there are two ways to trigger the Hooks: 


    this.on('mounted', function (evt) {
      /* ... */
    });

    this.on('loaded', function (evt) {
      /* ... */
    });

    this.on('destroyed', function (evt) {
      /* ... */
    });

Or, 


    this.mounted = function (evt) {
      /* ... */
    };

    this.loaded = function (evt) {
      /* ... */
    };

    this.destroyed = function (evt) {
      /* ... */
    };

For further information about Lifecycle Hooks and Component Registration, check the documentation: Registering Component.
