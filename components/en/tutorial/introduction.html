Welcome to the Bouer tutorial. This will provide you a everything you need to build fast and simple web applications easily.

You can also consult the Docs and/or the Playground where you can find some example and play with.

Prerequisites

The tutorial assume that you are already familiar with HTML, Javascript and CSS (Optional). We are going to show some examples the make your understanding easier. If you are totally new to front-end development, it might not be the best idea to get into to the lib as your first step... We advise you to be aware of them and then come back to continue.

What is Bouer.js?

Bouer is a javascript library for building user interfaces, and helps in web applications development, providing a synchronous interaction between user interfaces and Javascript data.

Understanding Bouer

In Bouer we have a main instance were we provided the options of the app and components instances. A component is a reusable block of code that encapsulates HTML, CSS and JavaScript that belong together, it's written in .html.
