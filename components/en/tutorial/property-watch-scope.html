If we want to watch many properties at the same time, we use the app.react(/* ... */) method. 


    app.react(function() {
      /* ... */
    })

It reacts to changes of any property inside the scope, whenever a property changes it recalls the scope.

Like the watch method, we use the destroy method inside of the returned object to unwatch all the properties that are being watched: 


    var react = app.react(/* ... */);

    // destroying the watch processes
    react.destroy();
