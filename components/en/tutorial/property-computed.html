We can define a property that is computed from other properties, to achieve that we use a function with $computed: 


    // Descripted `get` and `set`
    data: {
      property: function $computed() {
        return {
          get(): any,
          set(value: any): void,
        }
      }
    }

Or, we can use the computed property with the inferred get: 


    // Inferred `get`
    function $computed() {
      return value;
    }

Fortunately, all the property used inside of the $computed function listened, meaning that the computed property should be always updated.
