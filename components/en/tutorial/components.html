We use components to reuse snippets of html code, in Bouer components are built with .html file and registered in options.components: 


    var app = new Bouer('selector', {
      components: [
        { /* ... */ }
      ]
    })

We use the component.template property to define the component html raw and component.path to specify where the component template is: 


    components: [
      {
        name: 'user-form',
        template: `
          

            
          
`
      }
    ]
      

    components: [
      {
        name: 'user-form',
        path: '/components/user-form.html'
      }
    ]

Composition

In Bouer components are composed by a root element, scripts, and styles.

To be able to export data from the component script to the component element we use this.export(/* ... */) method: 


    this.export({
      /* ... */
    });

By the default, component styles are global, to be able to restrict it to the component we use the scoped attribute.
