Given a form element, we can easily convert from a form to an object literal using app.toJsObj(/* ... */) method: 


    

      
      
    

      

    var formObj = app.toJsObj('#post-form');

We are able to build an object with children by adding the e-build annotation to the element that contains the child properties:

In case of explicit array build type we use the e-array annotation on the element we want to be interpreted as array:

The element conversion is not limited only to form element, we can use a div element.

By default, Bouer targets all the elements having the name attribute and try to get the value in value attribute.

We can change the target element attribute or add multiple targets for names and values for the conversion process. For further info check: toJsObj in Methods.
