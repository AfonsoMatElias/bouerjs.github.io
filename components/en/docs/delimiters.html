Delimiters

By default Bouer brings some delimiters, which one of them is used by Vue.js, Angular.js, and others.

The delimiters open a space in HTML to execute some javascript code, it means that we can execute any pure js code variables, function, ternary operatiors, etc. It only need to return a value.

Whenever data is an Object it turn it to JSON.

Common

To use this delimiter just do the following: {{ [Content] }}

HTML Delimiter

To use this delimiter just do the following: {{:html [Property] }}
Which is used to process html snippets passed by a property.

We can check all the available delimiters in property app.$delimiters.get(). These values is unalterable by this property.

To be able to define our own delimiter, we use the method app.$delimiters.add({ ... }), it expects an object having the following structure: 


        var app = new Bouer('#app');
        app.$delimiters.add({
          /** The name of the delimiter */
          name: string,
          /** The delimiter structure */
          delimiter: {
            /** The open syntax */
            open: string,
            /** The close syntax */
            close: string
          },

          /** The action that will the performed */
          action?: (valueToSet: string, node: Node, data: object): string { }
        });
